# Fake Engagement Detection using Behavioural Analytics

## Problem Statement
Social media platforms often experience fake engagement generated by bots or coordinated activity.

## Objective
To detect artificial engagement patterns using behavioural signals.

## Dataset
Synthetic dataset generated using statistical simulation.

Number of Records: 1000

## Features Used
- Post Gap
- Engagement Burst
- Comment Similarity
- Timing Regularity
- Burst Ratio
- Risk Index

## Model
Random Forest Classifier

## Output
- Bot Probability
- Authenticity Score

## Accuracy
~95%

## Insight
Bots tend to show:

- High engagement bursts
- Predictable timing
- Similar commenting patterns

However, behavioural overlap exists between humans and bots.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn
